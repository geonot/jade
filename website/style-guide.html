Jinn Brand Guide

Identity system for the Jinn programming language

1. Brand Essence

Jinn is a systems language that marries the clarity of scripting with the performance of C. The brand draws from the cultural significance of jinn stone in East Asian philosophy — representing virtue, clarity, precision, and enduring value.

Core Attributes

  • Clarity — Code reads like clear English. No cryptic symbols where words suffice.
  • Precision — Every value is its type. No wrappers, no ambiguity.
  • Speed — 0.94× Clang -O3. Performance is verified, not promised.
  • Elegance — Minimal syntax, maximal expressiveness. Wu Wei — effortless action.
  • Endurance — Like jinn stone, built to last. Deterministic memory, no GC pauses.

Voice & Tone

Confident but understated. Technical authority conveyed through precision, not volume. Sentences are short. Code speaks first. Like a master calligrapher — every stroke intentional, nothing wasted.

4. Typography

Display — Cormorant Garamond

An elegant serif with classical proportions, used for all headings, titles, and pull quotes. Its refined character echoes the precision of jinn carving. Available in regular, medium, semibold, and bold weights; italic for emphasis.

Display / Headings
Reads like poetry.
Runs like lightning.

Body — Inter

A clean, highly legible variable sans-serif optimized for screens. Designed for UI at small sizes. Weights 300–700. The neutral character lets the content speak.

Body / UI Text
Jinn compiles through LLVM 21 to native code that matches Clang -O3. No runtime. No GC. No 64-byte Value class. Every integer is a register. Every class is contiguous memory. Every function is a native call.

Code — JetBrains Mono

A developer-focused monospace with increased height for readability and subtle distinctions between similar glyphs (0/O, 1/l/I). Used for all code samples, inline code, and terminal output.

Code / Terminal
*fib(0) is 0
*fib(1) is 1
*fib n is fib(n - 1) + fib(n - 2)

7. Writing Principles

  • Show, don't tell. Code examples first. Explanations second. Let the syntax speak.
  • Concise over verbose. Every sentence earns its place. Cut words that don't add meaning.
  • Technical precision. "0.94× Clang -O3" not "really fast." Numbers over adjectives.
  • Understated confidence. No superlatives. No "revolutionary." State facts. Let readers conclude.
  • Eastern aesthetic. 玉不琢，不成器 — "Uncut jinn is not a vessel." Mastery requires craft.
